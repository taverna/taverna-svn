Main help Menu options Operations

Exit

Exits the program. The application will attempt to close each open file as if you pressed "Close" for each of them. The exit operation will be aborted if any of the files being closed fails.

Operations

Exiting the program

  1. Click the "File" menu.
  2. Click on the "Exit" button. You may be prompted to save changes to a number of files. If any close operation fails the program will not exit until you've addressed the problem described to you.
