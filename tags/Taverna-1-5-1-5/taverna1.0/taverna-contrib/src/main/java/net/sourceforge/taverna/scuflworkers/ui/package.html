Provides LocalWorkers capable of getting input from a user, or providing feedback to a user. These classes use the JOptionPane class to create dialog boxes.
  • AskWorker: Allows the user to provide text input in response to a question.
  • ChooseWorker: Allows the user to select a value from a list of radio buttons.
  • SelectFileWorker: Allows the user to select a file from the local file system.
  • SelectWorker: Allows the user to select a value from a listbox of options.
  • TellWorker: Displays a message (no user input).
  • WarnWorker: Display a warning message (no user input).
