Main help Menu options Operations

Save Template

Allows you to save the currently selected tree record and optionally all its subrecords as a template file. A template file can then be imported into another experiment file pressing the File menu's Open Template button

Operations

Saving a template

  • Select an item in the tree. It should appear highlighted in green and the record should appear in the right hand panel.
  • Click the "File" menu.
  • Click the "Template" submenu.
  • Click the "Save Template" button. A file selection dialog should appear. Choose a name for the template.

See also Opening a template
