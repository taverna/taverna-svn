Tom's Generic Ant Template

tmo@ebi.ac.uk, 27th March 2003

Readme for the generic build template. This is assuming you have some modern version of windows as your build OS, so either Win2000 or WinXP Pro. It may only work under recent java versions, I haven't tested it on anything prior to jdk1.4.1 and ant1.5.1.

Configuring the build

You will probably want to configure the project name and version, these are located fairly near the top of the build.xml file. I would strongly suggest that you read the entire file and understand it before doing something that destroys your code by mistake. You might also want to change or remove the splash screen, as it's kind of myGrid specific, but that's up to you. In addition, the default build file assumes you have the jikes compiler in your path, if you don't then you need to change the compiler value to 'modern'.

Default build target - 'ant all' or 'ant'

By default, running 'ant' in the project directory will compile, document (javadoc, not doxygen) and produce a jar file. This is equivalent to 'ant jar javadoc'.

AspectJ, Java and RMI compilation - 'ant java'

By default the compilation stage will use AspectJ to pre-process any java class, interface and aspect defininitions in your src directory, resulting in a build source tree in build/src, which is then compiled to build/classes. In addition, the RMI compiler is invoked on anything that is an appropriate target within the build/classes directory to compile the remote stub and skeleton classes.

Import Tidy - 'ant tidy'

This task operates directly on your source tree! Use it with caution. It will clean up the import statements at the head of each .java file.

Doxygen - 'ant doxygen'

In order to be able to use Doxygen from here you will need to install both Doxygen itself and also dot.exe and its associated libraries.

  • Dot and support libraries - need to be put on your PATH somewhere.
    • http://www.research.att.com/sw/tools/graphviz/dist/dot.exe
    • http://www.research.att.com/sw/tools/graphviz/dist/thirdparty.zip
  • Doxygen, you need version 1.2.18
    • ftp://ftp.stack.nl/pub/users/dimitri/doxygen-1.2.18-setup.exe

When you have these prerequisites, running the Doxygen task will generate HTML documentation in the build/doxygen folder.
