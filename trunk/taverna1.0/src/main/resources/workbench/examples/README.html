Example workflows

These are just a handful of the possible workflows you can build using Taverna. To open a workflow from inside Taverna, use File -> Open workflow.. and navigate to this directory.

Note that since most of these workflows depend on external web services provided by third parties, and occasionally some of these services might change or be unavailable, you might experience problems executing a few of the workflows. Sign up for the taverna-users mailing list to check for any updates on service disruptions.

You can find many more Taverna workflows contributed by users on the website myExperiment.org - in particular you should be able to find an updated list of these examples in the pack Taverna 1.7.1 starter pack. You can also see the list of workflows tagged as example by other users.

For more information about how to build your own workflows, see the Taverna user manual.
