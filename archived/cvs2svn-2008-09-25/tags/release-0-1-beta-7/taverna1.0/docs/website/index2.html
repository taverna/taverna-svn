About the project

The Taverna project aims to provide a language and software tools to facilitate easy use of workflow and distributed compute technology within the eScience community. As a component of the EPSRC funded myGrid project, Taverna is available freely under the terms of the LGPL.

Taverna is a collaboration between the European Bioinformatics Institute (EBI), IT Innovation and the Human Genome Mapping Project Resource Center (HGMP). Development is coordinated through the facilities generously provided by SourceForge.net and driven by the requirements of biologists from the Newcastle Centre for Life, amongst others.

For more information, or to download the language specification, object model and associated software, please select an appropriate item from the menu on the left of this page.

Quickstart

To get playing with Taverna as quickly as possible we suggest you download the binary version of the workbench (assuming that you are either on a win32 machine or that you are prepared to fiddle with the launch script). This will give you a chance to build, edit and run workflows using the embedded workflow enactment engine, see the workbench documentation in the menu to the left for more details.

What's new in beta7?

In this beta release we've added support for local services to perform operations such as set union and intersection, fetch from URL, list filtering based on regular expressions and similar. In addition, the implicit iteration support that was removed for the new data model introduction is back, enabling automatic iteration over data sets - wherever possible Taverna will now automatically reconcile type cardinality mismatches and hopefully behave sensibly. The level of complexity possible in workflows is significantly higher than previous releases due mainly to these new features, there's a PDF file available that shows a couple of examples of what is possible!
