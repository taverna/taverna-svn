Main help Menu options Operations

Save

Saves all changes made since the last time you pressed Save or Save As....

If you have created a record or have changed an entry in an existing record and pressed "Keep", you can observe your changes. See "Save As...", you should see the blue coloured markers disappear.

Operations

Save changes to file

  1. Click on the "File" menu.
  2. Click on the "Save" button. The operation will stop if the current record contains errors.
