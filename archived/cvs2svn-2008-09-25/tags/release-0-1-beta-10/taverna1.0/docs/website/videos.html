scufl workbench video clips

Version 0.1.beta7, 2nd December 2003, tmo@ebi.ac.uk

AVI format video clips from a workbench session

These video clips were created using the Camtasia Studio software from TechSmith, you may need to install the video codec in order to be able to play them, more information is available from http://www.camtasia.com.

The link below shows the most sophisticated workflow we have created thus far running, including the new result browser in action, and is taken from the beta7 release. Older videos (beta4 release) are below.

  • video3.avi - enactment of the SNP based annotation pipeline workflow, see the PDF linked from the home page for a more detailed view of this workflow. (4,540kb, 2:30 minutes run length)

The links below will take you to a couple of video clips of example runs of the SCUFL workbench, the first is a full cycle from loading a predefined workflow from the web, adding new functionality to it, enacting it on some input data and then saving the modified version. The second is much shorter and highlights the heirarchical service structure and descriptions provided by the Soaplab server at the EBI. These clips are from the beta4 release of Taverna.
