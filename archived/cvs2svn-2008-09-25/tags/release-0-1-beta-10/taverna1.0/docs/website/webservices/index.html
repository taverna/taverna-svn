Bioinformatics Web Services

tmo@ebi.ac.uk, 2nd December 2003

This page provides links to various services in the wild. Note that these are examples, and may or may not be available at any given time. In order to access these services from Taverna, you can either add the WSDL links explicitly using the 'Create new WSDL Scavenger' option in the service panel, or you can point the 'Collect Scavengers From Web' option at this page and it will attempt to load all files linked from here (this may take some time, but if you expand the web crawl node it should give you some idea of the progress of the operation). In the long run, we'd far rather these services were accessed from some kind of UDDI based registry, but for now this is a reasonable way to provide links to them!

Example workflows are provided in the 'samples' directory of the workbench distribution, but are mirrored here in order that the web crawler can find them as well.
