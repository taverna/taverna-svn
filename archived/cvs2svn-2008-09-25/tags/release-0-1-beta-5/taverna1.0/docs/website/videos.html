scufl workbench video clips

Version 0.1.beta4, 18th June 2003, tmo@ebi.ac.uk

AVI format video clips from a workbench session

The links below will take you to a couple of video clips of example runs of the SCUFL workbench, the first is a full cycle from loading a predefined workflow from the web, adding new functionality to it, enacting it on some input data and then saving the modified version. The second is much shorter and highlights the heirarchical service structure and descriptions provided by the Soaplab server at the EBI.

  • video1.avi - complete editor run (4,664kb, 4:28 minutes run length)
  • video2.avi - simple demo of soaplab functions (1,263kb, 1:20 minutes run length)
