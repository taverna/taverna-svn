Contains Interaction Patterns and support code for Genome flatfiles.

This package contains Interaction Patterns for handling Genome flatfiles (such as EMBL, Genbank, GFF etc.). The only editor supported for these files so far is a modified version of Artemis launched via Java Web Start for manipulation of flatfiles.

The expert reviewer (user) is presented with the following data:

  • Title (of the file / list of files)
  • Comments
  • Annotation data (the file content itself)

After automatically downloading and reviewing the annotation data using an editor, the following data is returned from the reviewer :

  • Decision (either "Accept" or "Reject")
  • Modified ("yes" or "no" depending on whether the data has been modified)
  • Review Notes
  • Reviewed data (either the original annotation data or the modified data)
