Favourite File

Opens a favourite file. Each time you start the application, it will present a list of most recently used files under the "Favourites" sub menu of the "File" menu.

You will not see this File menu feature when you use the program for the first time. You must open existing files or save new ones for them to be registered as a favourite file the next time you start the application.

Operations

Open a favourite file

  1. Click on the "File" menu.
  2. Select the "Favourites" submenu. See the description above if you don't see the submenu appear. You should see a list of files that you've recently been using.
  3. Select one of the buttons showing the name of a file you'd like to open. A new dialog should appear with the experiment file you selected.
