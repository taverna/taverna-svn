Search

Causes a search dialog to appear. The search utility scans the names of subrecords in the tree which match what you type in the search field. The search field allows you to specify special characters make your search more specific. To search:

  1. Type a regular expression in the search field.
  2. Press "Apply". Those subrecords in the experiment file whose name matches the search field expression should appear highlighted in purple in the tree.

Selecting an item in the search list causes that record to appear in the experiment file window.
